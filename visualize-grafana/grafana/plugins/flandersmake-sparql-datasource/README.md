# Grafana SPARQL Datasource Plugin

![Catalog](https://img.shields.io/badge/dynamic/json?logo=grafana&query=$.version&url=https://grafana.com/api/plugins/flandersmake-sparql-datasource&label=catalog&prefix=v&color=F47A20)
![Downloads](https://img.shields.io/badge/dynamic/json?logo=grafana&query=$.downloads&url=https://grafana.com/api/plugins/flandersmake-sparql-datasource&label=downloads&color=F47A20)

This datasource plugin adds support for [SPARQL](https://www.w3.org/TR/sparql11-query/) endpoints to [Grafana](https://grafana.com/). Under the hood, it uses the [Comunica](https://comunica.dev/) framework.
